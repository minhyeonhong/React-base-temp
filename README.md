<br/>

## 설치한 라이브러리들
- styled-components

        yarn add styled-components

- redux toolkit

        yarn add redux
        yarn add react-redux @reduxjs/toolkit

- react-router-dom

        yarn add react-router-dom

- json-server

        yarn add json-server

- axios

        yarn add axios

- cookie

        yarn add react-cookie

- browser-image-compression

        npm i browser-image-compression
